建議老師優先使用這兩種

Option 1

Google Gemini API Key

適合一般老師先使用。Google AI Studio 可建立 Gemini API key，部分模型有免費額度。

  1. 開啟 Google AI Studio API Keys 頁面。
  2. 用自己的 Google 帳號登入。
  3. 點選 Create API key。
  4. 複製產生的 API key。
  5. 回到產生提示詞頁，Provider 選 Google Gemini，貼上 API key。
  6. 模型清單會即時載入，選一個模型後開始產生內容。
前往 Gemini API Key 官方說明

Option 2

OpenRouter API Key

適合想使用多種免費模型的老師。OpenRouter 有免費模型集合，但模型速度、限制與可用性可能變動。

  1. 開啟 OpenRouter 登入頁。
  2. 用 Google、GitHub 或 Email 建立帳號。
  3. 進入 API Keys 頁面建立新的 key。
  4. 複製 API key。
  5. 回到產生提示詞頁，Provider 選 OpenRouter，貼上 API key。
  6. 模型清單會即時載入，選擇免費或可用模型後開始產生內容。
前往 OpenRouter 查看免費模型
API key 請老師自行保管，不要貼到公開文件、社群訊息、簡報截圖或學生可看到的頁面。本網站不會把 API key 寫入檔案。
